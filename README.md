# Fashion Popularity Prediction ML Project

A Machine Learning project built in Google Colab using Python, Pandas, Scikit-learn, and Matplotlib to analyze fashion product data, predict popularity scores, and perform clustering analysis.

This project explores both:

- **Supervised Machine Learning**
- **Unsupervised Machine Learning**

using a fashion dataset containing pricing, style, engagement, and visual attributes.

---
#  Project Objectives

The main goals of this project are:

- Predict fashion item popularity using machine learning models
- Compare supervised learning algorithms
- Evaluate model performance using accuracy and R² score
- Perform customer/product segmentation using clustering
- Analyze fashion-related features such as:
  - Brightness
  - Colorfulness
  - Comfort score
  - Views
  - Likes
  - Seasonal wear
